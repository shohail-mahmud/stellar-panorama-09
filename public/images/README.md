
# Planet Images

This folder should contain transparent PNG images of planets for the solar system website.

Required images:
- sun.png
- mercury.png
- venus.png
- earth.png
- moon.png
- mars.png
- jupiter.png
- saturn.png
- uranus.png
- neptune.png

For best results:
- Use transparent PNG format
- High resolution (at least 1000x1000 pixels)
- Consistent lighting angle
- Clean transparent background
- Include rings or distinctive features (for planets like Saturn)

Example sources:
- NASA image libraries
- Space agency websites
- High-quality astronomy resources
